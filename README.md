# AlphaShapePackage
An R package to compute alpha shapes and visualize 2D and 3D geometric structures for bifurcation analysis

Dependencies:
    dbscan,
    sf,
    sp,
    transport,
    testthat

## Installation

Install the released version from CRAN:

```r
install.packages("AlphaShapePackage")
```

```r
# install.packages("remotes")
remotes::install_github("tlstellGSU/AlphaShapePackage")
```
